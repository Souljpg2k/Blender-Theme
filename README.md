## Installation
1. Download this [ZIP File](https://github.com/Souljpg2k/Blender-Theme/archive/refs/heads/main.zip) and extract.
2. In Blender go to `Edit` > `Preferences` > `Themes`
3. Click the **Install...**
4. Select the `MYC.xml` file and click **Install Theme**.
